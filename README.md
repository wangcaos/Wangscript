# WangScript v2.4 - Roblox Universal Hack GUI

WangScript là script hack Roblox đa năng, hỗ trợ GUI đẹp với pill switch mượt, bind phím, slider chỉnh tay (FOV, Speed, Range), vuông xanh lock-on aimbot (click toggle), KillAura, Illusion shadow (Naoya JJK style), Semi Bypass, lệnh chat `.fly`, `.aimbot`...

**Cảnh báo**: Vi phạm TOS Roblox → dùng alt acc, tự chịu rủi ro ban HWID!

## Tính năng chính
- Fly (điều chỉnh speed tay)
- Speed Hack (điều chỉnh WalkSpeed)
- Noclip
- ESP & Trace
- Aimbot (vuông xanh lock-on khi click LMB toggle)
- Silent Aim
- Invisible FE
- KillAura (range slider)
- Illusion Shadow (bóng ảo ảnh như Naoya JJK khi di chuyển)
- Semi Anti-Cheat Bypass
- Lệnh chat prefix `.` (`.help` để xem)
- GUI tab ngang (Combat, Movement, Player, Misc)
